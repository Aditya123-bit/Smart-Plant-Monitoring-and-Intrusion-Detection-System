# 🌱 Smart Plant Monitoring and Intrusion Detection System

An IoT-based smart system that monitors plant health conditions and detects intrusion using ESP8266, sensors, and a web dashboard.

---

## 🚀 Features

- 💧 Soil Moisture Detection
- 🚨 Intrusion Detection using PIR Sensor
- 📟 Real-time Data Display on LCD/LED
- 🌐 Web Dashboard for Monitoring
- 📡 ESP8266 WiFi Connectivity
- 🔔 Alert Notification System
- 📱 Responsive Frontend Design

---

# 🛠️ Tech Stack

## Hardware Components
- ESP8266 NodeMCU
- Soil Moisture Sensor
- PIR Motion Sensor
- DHT11 Sensor
- 16x2 LCD Display
- Breadboard
- Jumper Wires

## Software Technologies
- HTML
- Tailwind CSS
- JavaScript
- Node.js
- Express.js
- JSON Database
- Arduino IDE
